# 📝 Task Manager CLI (Java)

Prosta aplikacja konsolowa napisana w Javie do zarządzania zadaniami (To-Do).
Projekt stworzony w celu nauki podstaw programowania obiektowego oraz pracy z danymi.

---

# 🚀 Funkcjonalności

* ➕ Dodawanie nowych zadań
* 📋 Wyświetlanie listy zadań
* ✅ Oznaczanie zadań jako ukończone
* ❌ Usuwanie zadań (zakończenie taska)
* 🔍 Filtrowanie:

  * tylko ukończone
  * tylko nieukończone
* 🔤 Sortowanie zadań po tytule
* 💾 Trwały zapis danych do pliku (`tasks.txt`)

---

# 🧠 Technologie i koncepty

* Java (OOP)
* Kolekcje (`ArrayList`)
* Stream API (filter, forEach)
* Operacje na plikach (File I/O)
* CLI (Command Line Interface)

---

# 🏗️ Struktura projektu

* `Task` – model danych reprezentujący zadanie
* `TaskManager` – logika biznesowa (zarządzanie zadaniami)
* `Main` – interfejs użytkownika (CLI)

---

# ▶️ Jak uruchomić

1. Skompiluj projekt:

```
javac Main.java
```

2. Uruchom aplikację:

```
java Main
```

---

# 📌 Przykładowe użycie

```
1. Add task
2. List tasks
3. Complete task
4. Delete task
5. Show completed
6. Show pending
7. Sort by title
8. Exit
```
